# Flujos

## Flujo de login

1. Usuario abre `/login`.
2. Ingresa usuario y contrasena.
3. El backend valida contra `.env`.
4. Si es correcto, crea sesion y redirige a `/panel`.
5. Si falla, muestra mensaje amigable.
