# UI/UX

## Lineamientos visuales

Estilo moderno, sobrio, corporativo y futurista moderado. Debe priorizar lectura, control operativo y claridad de estados.

## Paleta de colores

* Blanco predominante para superficies.
* Azul corporativo para acciones primarias.
* Azul oscuro para sidebar.
* Cyan moderado para acentos.
* Verde para exito.
* Rojo para errores.
* Amarillo/naranjo para advertencias.
* Grises suaves para bordes y textos secundarios.

## Tipografias sugeridas

`Segoe UI`, Arial o sans-serif del sistema para mantener rendimiento y compatibilidad en Windows y Linux.

## Componentes iniciales

* Login centrado.
* Sidebar.
* Topbar.
* Tarjetas de metricas.
* Alertas.
* Botones primarios y secundarios.

## Layout

Aplicacion con sidebar lateral en escritorio y layout apilado en pantallas menores.

## Estados visuales

La Fase 1 define colores base. Estados funcionales completos se implementaran con tareas, ejecuciones y logs.

## Reglas de experiencia

* Acciones criticas deben confirmar en fases posteriores.
* Errores deben ser claros y accionables.
* Los paneles deben permitir escaneo rapido.
